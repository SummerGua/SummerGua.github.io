# 父子组件问题

在父组件中使用一个封装好的el-rate组件(此时名为Stars)时，修改星星数量会导致报错，是因为子组件局部值和props值都被修改，这种修改props值的方式是不被推荐的，需要在子组件中使用如下代码：

data(){
  return{
    stars:0
  }
}
updated(){
  this.$emit('star',this.stars)
}

父组件如下调用：

<Stars @star="getStar"></Stars>

data(){
  return{
    stars:0
  }
},
methods:{
  getStar(data){
    this.stars = data
  }
}
